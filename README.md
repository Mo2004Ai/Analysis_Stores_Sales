Overview

This project focuses on analyzing sales data to understand key factors affecting sales and profitability, with the goal of improving net profit. The project includes an interactive dashboard that consolidates all visualizations for easy comparison and insight generation.

Approach

Data Understanding

Explored the dataset to understand the features and overall structure.

Checked for outliers, missing values, and any anomalies in the data distribution.

Data Analysis

Used groupby operations to examine relationships between features.

Created visualizations to gain deeper insights into patterns and trends in sales and profit.

Dashboard

Developed a comprehensive dashboard that aggregates all visualizations in one place.

Allows for easy comparison between charts and a clear view of actionable insights.

Key Skills & Tools

Python (Pandas, Matplotlib, Seaborn, Plotly)
